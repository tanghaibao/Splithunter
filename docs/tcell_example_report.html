Mask semantics. The V-J focal window is intronic in every reference build, so this estimator measures the off-target bleed depth — reads that spill from captured exons into the surrounding intron and drop when the V-J region is deleted in T-cells. On WES, the few positions that fall inside captured exons carry bait-inflated depth (~40× vs ~2× bleed) and can dominate the baseline median while the focal median stays on the bleed scale, biasing the log-ratio. Splithunter masks exon positions (passed via --targets BED) so focal and baseline medians are taken on the same bleed-depth scale.

On cov_example the headline fraction is unchanged: the focal window is entirely intronic (so all 30,133 focal positions survive), and exon positions make up only ~5% of the baseline window (11,589 / 237,219 dropped). The mask is a guard against bait-edge bias rather than the source of the signal on this pre-cleaned demo dataset — on a raw WES BAM with high-depth exon spikes, the effect is material.

Interpretation

  • Strong, statistically clean dip. Focal median depth (1) is exactly half the baseline median (2), giving log2R = -1.0 and an estimated 50% T-cell fraction. The CI is bounded at 0.5 from above because the focal depth cannot drop below 1 read in this exome dataset.
  • Biological mechanism. Mature T-cells delete one allele of TCRA during V-J recombination; in a sample where ~50% of cells are T-cells, ~25% of TCRA-locus DNA is missing at the deletion midpoint, yielding the focal coverage drop visible above.
  • Contrast with NA12878 (TRA splithunter report). The NA12878 LCL slice in tests/NA12878.bam shows focal/baseline medians of 29/31 (log₂R ≈ -0.10, fraction ≈ 6% with CI [0%, 71%]) — i.e. no real signal, as expected for an EBV-immortalized B-cell line.

Reproduce

python -c "
import csv, gzip
from splithunter import _core
from splithunter.loci import load_exons_bed
segs = {r['segName']: (int(r['start']), int(r['end']))
         for r in csv.DictReader(open('tests/fixtures/tcra_seg_hg19.csv'))}
pos, dep = [], []
for r in csv.DictReader(gzip.open('tests/fixtures/cov_example.csv.gz', 'rt')):
    pos.append(int(r['pos'])); dep.append(int(r['reads']))
mask = load_exons_bed('tests/fixtures/tcra_exons_hg19.bed', 'chr14')
print(dict(_core.fraction_from_coverage(
    pos, dep,
    segs['focal'][0], segs['focal'][1],
    segs['local1'][0], segs['local1'][1],
    segs['local2'][0], segs['local2'][1],
    0,                       # min_cov
    mask)))                  # target exons to exclude from the bleed median
"

On a raw hg38 WES BAM, pass --targets kit.bed to splithunter_run or splithunter_tcell; for TRA on hg38 the packaged splithunter/data/tcra_exons_hg38.bed is used as the default mask.
