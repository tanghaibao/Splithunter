The corrected default is the more faithful TRA/WES interpretation: the focal window is intronic, so the upstream-style run has no focal exon positions and therefore reports an effectively null T-cell signal for NA12878. That is more consistent with NA12878 being an EBV-immortalized B-cell line. The older whole-window shortcut exaggerated a weak intronic coverage difference into a non-zero fraction.
